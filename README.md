# Tracery Demo

This is a demo of Kate Compton's [Tracery](https://github.com/galaxykate/tracery), a Javascript text-expansion library.

Run:

```
$ python -m SimpleHTTPServer 8081
```

Visit http://localhost:8081/ to see the demo.
